AddCardViewController

public class AddCardViewController : UIViewController, ValidationDelegate

View controller to manage a credit card entry. It renders a ‘Cancel’ and ‘Add card’ buttons so it must be presented inside a UINavigationController.

Presenter helper can be used to show the View Controller.

User can fill out the form and on submission card data will be provided to addCard delegate. User can cancel the form and proper delegate call is performed.
